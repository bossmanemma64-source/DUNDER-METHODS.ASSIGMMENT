# Dunder Methods Assignment
This project shows Python dunder/special methods using simple classes.

## Dunder Methods Implemented

1. `__getitem__` - Access items like list/dict  
   Example: s1["Math"]

2. `__setitem__` - Assign items like dict  
   Example: s1["Math"] = 95

3. `__contains__` - Use `in` keyword  
   Example: "Python" in lib

4. `__call__` - Make object callable  
   Example: counter()

5. `__iter__` and `__next__` - Make object iterable  
   Example: for x in countdown

6. `__bool__` - Define truth value  
   Example: if student:

7. `__del__` - Cleanup when object deleted  
   Example: del file
